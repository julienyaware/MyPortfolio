Term Project

LLM Educational Assistant

This semester, my team and I will develop a web application that will use Large Language Models (LLMs) to summarize long complex educational content into simplified, shortened summaries.

The goal of the project is to create a tool that will help students and professionals to grasp information quickly without having to read through long and complex content.

Some of the main features of the application include a user-friendly interface that will allow the user to paste long content and view the output, a summarization functionality, and generating quizzes to gauge the understanding of the user.

We will use Python and Langchain for the back-end, Firebase for the login and authentication and React and Tailwind CSS for the front-end.
